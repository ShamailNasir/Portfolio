# Portfolio

Live website: [shamailnasir.dev](https://shamailnasir.dev).

## Architecture & Tech Stack
- **Frontend**: HTML5, Vanilla CSS, React (compiled and run client-side)
- **Deployment**: Vercel

## Key Features
- **Performance-optimized embeds**: Interactive project previews render in HD using CSS layout zoom, and load dynamically using `IntersectionObserver` to unmount off-screen previews and eliminate page lag.
- **Interactive UI**: Custom cursor tracking, modular section layouts, and interactive canvas components.
- **Asset Manifest**: Bundled assets, scripts, and fonts are compiled directly inside the HTML payload.

## Structure
- `index.html`: Entry point containing the base markup, styles, React components, and asset manifest.
